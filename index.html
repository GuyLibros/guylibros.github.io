What is Interlibrary Loan (ILL)?

Interlibrary Loan (ILL) is a service that allows students to borrow resources from other libraries worldwide.

How to Access ILL Services

Visit our library website to set up an account and start using ILL services. Visit the ILL Page

Requesting Materials

Follow these steps to request books, articles, and more:

  1. Log into your ILL account.
  2. Select the material type and complete the request form.
